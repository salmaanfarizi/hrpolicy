# HR Policy Manual - ARS International

Multi-language HR Policy Manual for ARS International Co. S.P.C.

## Languages Available
- English
- Arabic (العربية)
- Hindi (हिंदी)
- Malayalam (മലയാളം)
- Bengali (বাংলা)
- Nepali (नेपाली)

## Deployment
This site is deployed on Netlify. Simply push to the main branch to deploy.

## Files
- `index.html` - Language selector landing page
- `english.html` - English manual
- `arabic.html` - Arabic manual (RTL)
- `hindi.html` - Hindi manual
- `malayalam.html` - Malayalam manual
- `bengali.html` - Bengali manual
- `nepali.html` - Nepali manual
- `netlify.toml` - Netlify configuration
